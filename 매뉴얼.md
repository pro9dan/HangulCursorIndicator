# Hangul Cursor Indicator 매뉴얼

## 프로그램 개요

- (한영키 알림 기능) Hangul Cursor Indicator는 Windows에서 현재 한/영 입력 상태를 마우스 포인터 근처에 표시하고, 
- (익명 문자 방송) 같은 LAN에서 실행 중인 사용자에게 일회성 익명 토스트 메시지를 보낼 수 있는 데스크톱 프로그램입니다.

## 왜 만들었나요?

- 우리는 오래된 불편함에 익숙해져 버렸다.
- (한영키 알림 기능) 윈도우에서 기본적으로 한영키의 전환이 불안전하고 매번 입력창에 들어왔을때 한영키의 상태를 알기 쉽지 않음
- (익명 문자 방송) 익명 게시판이 있지만 이것도 기록이 남고 접근하기가 사실 부담스러움, 그냥 익명으로 실없는 소리 날리고 싶을때가 있음

## 실행 방법

- /dist 경로에서 배포받은 `HangulCursorIndicator.exe`를 실행하면 프로그램은 백그라운드에서 동작하며 시스템 트레이에 아이콘이 표시됩니다.
- 설치가 필요없고 standalone으로 바로 실행되어 접근성이 편합니다.

## 한/영 표시 기능

- 한글 입력 모드이면 마우스 포인터 근처에 `한` 배지가 표시됩니다.
- 영문 입력 모드이면 마우스 포인터 근처에 `A` 배지가 표시됩니다.
- 배지는 클릭을 방해하지 않습니다.
- 일부 프로그램은 자체 입력기를 사용해 한/영 상태가 정확히 감지되지 않을 수 있습니다.

## 익명 문자 발송 기능

1. `Ctrl+Shift+P`를 누릅니다.
2. 현재 모니터 중앙에 입력창이 표시됩니다.
3. 메시지를 입력합니다.
4. `Enter`를 누르면 같은 LAN에서 이 프로그램을 실행 중인 다른 사용자에게 메시지가 전송됩니다.

취소 방법:

- `Esc` 키를 누릅니다.
- 입력창 오른쪽의 `X` 버튼을 누릅니다.
- 입력창 밖으로 포커스가 이동하면 입력창이 닫힙니다.

## 메시지 수신

- 수신 메시지는 현재 모니터 중앙에 토스트처럼 표시됩니다.
- 몇 초 후 자동으로 사라집니다.
- 발신자 정보는 표시하지 않습니다.
- 메시지는 파일이나 데이터베이스에 저장하지 않습니다.
- 메시지를 보낸 본인에게는 토스트가 표시되지 않습니다.

## 시스템 트레이 메뉴

트레이 아이콘을 우클릭하면 다음 메뉴를 사용할 수 있습니다.

- 표시 켜기/끄기: 한/영 배지 표시를 켜거나 끕니다.
- 프로그램 시작 시 자동 실행 켜기/끄기: 현재 사용자 기준 Windows 시작 프로그램 등록을 켜거나 끕니다.
- 프로그램 정보: 개발자와 연락처를 표시합니다.
- 종료: 프로그램을 종료합니다.

## 자동 실행

자동 실행은 현재 사용자 레지스트리 Run 키를 사용합니다.

```text
HKCU\Software\Microsoft\Windows\CurrentVersion\Run
```

관리자 권한은 필요하지 않습니다.

## 로그 위치

문제 분석용 로그는 아래 폴더에 날짜별로 저장됩니다.

```text
%LOCALAPPDATA%\HangulCursorIndicator\logs
```

예:

```text
C:\Users\<사용자>\AppData\Local\HangulCursorIndicator\logs\20260611.log
```

## 네트워크 주의사항

- 메시지 전송은 UDP broadcast를 사용합니다.
- 기본 포트는 `45455`입니다.
- 같은 LAN 안에서만 동작합니다.
- Windows 방화벽 또는 네트워크 장비가 UDP broadcast를 차단하면 메시지가 전달되지 않을 수 있습니다.
- 암호화나 사용자 인증은 제공하지 않습니다.

## 배포 파일

일반적인 64비트 Windows 10/11 PC에는 아래 파일 하나만 배포하면 됩니다.

```text
HangulCursorIndicator.exe
```

self-contained single-file EXE이므로 대상 PC에 .NET 런타임을 별도로 설치할 필요가 없습니다.
